# 📊 Student Performance Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **Student Performance Dashboard** developed in **Microsoft Power BI**. It analyzes student academic performance, attendance, and behavior using multiple datasets and interactive visualizations.

The dashboard provides insights into student scores, attendance percentage, performance categories, and behavior records through dynamic reports.

---

## 📂 Dataset

The project uses the following CSV files:

- **Students.csv**
  - StudentID
  - Name
  - Gender
  - Class
  - Section

- **Scores.csv**
  - StudentID
  - Subject
  - ExamType
  - Score
  - MaxScore
  - Term

- **Attendance.csv**
  - StudentID
  - Date
  - Status (Present/Absent)
  - Reason

- **Behavior.csv**
  - StudentID
  - Date
  - BehaviorType
  - Notes

---

## 📈 DAX Measures

The following DAX measures were created:

- % Score
- Average Score per Subject
- Attendance %
- Behavior Count per Type
- Performance Category (High / Medium / Low)

---

## 📊 Dashboard Visualizations

- KPI Cards
  - Total Students
  - Average Attendance
  - Average Score
  - Performance Category

- Line Chart
  - Average Score per Subject by Term

- Bar Chart
  - Average Score by Class

- Behavior Visualization

- Student Performance Table
  - Student-wise scores
  - Conditional Formatting

- Slicers
  - Class
  - Exam Type
